# TaskPilot Overview

TaskPilot is an AI-powered project management platform designed for Agile/Scrum teams. It combines standard project operations with an intelligent assistant that understands natural language, executes business actions, and recommends assignees based on skills and workload.

## Core Goals

- Build a practical AI Agent for project teams.
- Improve planning and execution visibility.
- Reduce manual coordination effort in task assignment and tracking.

## Product Highlights

- End-to-end project and task management with structured task attributes.
- Interactive Kanban workflow: TODO -> IN_PROGRESS -> REVIEW -> DONE.
- Dashboard metrics for progress and delivery monitoring.
- AI chat interface for command-based task and workflow actions.
- JWT-based authentication and role-based access.

## Main Actors

- Project Manager: coordinates team execution and uses AI-assisted assignment.
- Member: updates work progress and collaborates through tasks/comments.
- Administrator: manages system configuration, users, and AI settings.
